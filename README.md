# 📊 Poll Results Visualizer

🚀 An end-to-end Data Analytics project that transforms raw poll/survey data into meaningful insights using Python and Streamlit.

---

## 📌 Overview

Poll data is widely used across industries for decision-making. This project simulates a real-world poll analytics system that:

* Processes survey data
* Performs analysis
* Generates visual insights
* Provides an interactive dashboard

---

## 🎯 Problem Statement

Organizations collect survey data but often lack tools to:

* Quickly analyze responses
* Understand trends
* Extract actionable insights

---

## 💡 Solution

This project builds a **Poll Results Visualizer** that:

* Analyzes poll responses
* Calculates vote share and trends
* Provides region-wise and demographic insights
* Displays results through an interactive dashboard

---

## ⚙️ Features

✔ Synthetic dataset generation (500+ responses)
✔ Data cleaning and preprocessing
✔ Vote share analysis (counts & percentages)
✔ Region-wise comparison
✔ Trend analysis over time
✔ Interactive filters (Region, Age Group)
✔ KPI metrics (Top choice, % share)
✔ Insights generation
✔ Download filtered data option
✔ Interactive Streamlit dashboard

## ▶️ How to Run

1. Clone Repository

```bash
   git clone https://github.com/Anand1745/Poll-Results-Visualizer.git
   cd Poll-Results-Visualizer
```

2. Create Virtual Environment

```bash
   python -m venv venv
   venv\Scripts\activate     (Windows)
   source venv/bin/activate  (Mac/Linux)
```

3. Install Dependencies

```bash
   pip install -r requirements.txt
```

4. Run Project

```bash
   python main.py
   streamlit run dashboard.py
```
